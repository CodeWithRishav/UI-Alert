Welcome to the Tutorial Page

Learn how to make the most of our project dashboard and tools.

Step 1: Adding a New Project

Navigate to the main dashboard and click on the "Add New Project" button. Fill in the project details such as title, description, and status. Save the project to see it listed on the dashboard.

Step 2: Editing an Existing Project

Click the "Edit" button on the project card you want to update. Modify the necessary details and save your changes. The updated details will appear instantly on the dashboard.

Step 3: Deleting a Project

If a project is no longer required, click the "Delete" button on the respective project card. Confirm the action, and the project will be permanently removed.

Tip: Be careful while deleting projects as this action is irreversible!

Step 4: Navigating to Tutorials

To revisit this page, click on the "Tutorials" link in the navigation menu. Use this page to review features and enhance your understanding of the platform.
